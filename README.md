## Overview

This repository contains the SAP GRC-Lite ruleset and compliance-as-code blueprint for Telin Group's SAP S/4HANA (Client 300) User Access Management (UAM) and User Access Review (UAR) program.

**Owner of DEFINITION:** GR Unit + IT Risk (2nd line)
**Owner of OPERATION:** IT (1st line)
**Assurance:** Internal Audit (3rd line)

## Standards & Frameworks

- **ICoFR Basis:** SK-5/DKU.MBU/11/2024 (COSO 2013)
- **Audit Standard:** PCAOB AS 2201 / AS 1105
- **Review Period:** 2026-01-01 to 2026-06-30
